# Agon Health — Colour Scheme

Warm, earthy palette with a vibrant orange accent.

| Token | Hex | Usage |
|-------|-----|-------|
| `--bg` | `#FAF3E9` | Page background |
| `--surface` | `#E8DAC5` | Cards, elevated surfaces |
| `--border` | `#D9CBB4` | Borders, dividers |
| `--text-primary` | `#3E2B23` | Headings, body text |
| `--text-secondary` | `#6B584C` | Subtitles, muted text |
| `--secondary` | `#A67B5B` | Secondary buttons, accents |
| `--accent` | `#FF5A1F` | Primary CTA, links, highlights |
| `--accent-pressed` | `#E64A0D` | Hover/active state for accent |
| `--accent-tint` | `#F2C8A8` | Light accent backgrounds, badges |

## CSS Variables

```css
:root {
  --bg: #FAF3E9;
  --surface: #E8DAC5;
  --border: #D9CBB4;
  --text-primary: #3E2B23;
  --text-secondary: #6B584C;
  --secondary: #A67B5B;
  --accent: #FF5A1F;
  --accent-pressed: #E64A0D;
  --accent-tint: #F2C8A8;
}
```
